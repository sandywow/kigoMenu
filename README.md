# 享·奇果 菜單網頁

咖啡廳數位菜單，純靜態網頁，無需伺服器，手機開啟即用。

## 檔案結構

```
menuWeb/
├── index.html      # 菜單主頁面
├── admin.html      # 後台設定頁面
├── config.js       # 菜單資料與設定
├── app.js          # 頁面邏輯
├── style.css       # 樣式
└── src/
    └── *.jpg       # 品項圖片
```

## 使用方式

### 日常更新菜單

1. 用瀏覽器開啟 `admin.html`
2. 在「菜單品項」分頁新增、編輯或移除品項
3. 在「首頁設定」分頁修改首頁文案、推薦卡、Banner 等
4. 按右上角「**儲存設定**」→ 暫存於瀏覽器（重整首頁即可預覽）
5. 確認無誤後按「**複製 config.js**」→ 貼上覆蓋 `config.js` 檔案以永久生效

### 直接編輯設定檔

開啟 `config.js`，修改 `menuData`（品項）或 `landingData`（首頁）後存檔即可。

## config.js 欄位說明

### 品項欄位（menuData）

| 欄位 | 說明 | 空白時 |
|------|------|--------|
| `name` | 品名 | 不顯示 |
| `nameJp` | 副品名（支援 `\n` 換行） | 不顯示 |
| `desc` | 說明文字（支援 `\n` 換行） | 不顯示 |
| `price` | 價格，如 `NT$150` | 不顯示 |
| `tag` | 標籤，如「限定」「每日限量」| 不顯示 |
| `image` | 圖片路徑或網址 | 不顯示圖片 |
| `temp` | `"hot"` / `"iced"` / `"both"` / 不填 | 不顯示溫度點 |

### 首頁設定（landingData）

各欄位設為 `null` 即隱藏該區塊。

### 分類索引鍵（tabs）

`key` 對應 `menuData` 裡的屬性名稱，`label` 為分頁顯示文字。

## 新增圖片

將圖片放入 `src/` 資料夾，在 `config.js` 的 `image` 欄位填入路徑，例如：

```js
image: "src/lemon.jpg"
```
